# **RZERVA.COM**

An online platform that connects travelers seeking an affordable and friendly place to stay with homeowners who have a spare room.

### **Softwares & Packages**

- Python
- Html
- Bootstrap
- Javascript
- CSS
- Jinja 2 
- Flask
- Sqlite3 
- Docker

### **Features**

- Html pages
- User Dashbord
- User authentication system
- Password reset with secure cookies
- Email Notification
- Admin Dashbord
